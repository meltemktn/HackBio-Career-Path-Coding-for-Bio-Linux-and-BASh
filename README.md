## Coding for Bio : Linux and BASh

This directory includes the scripts for the two projects under the HackBio Career Path Course titled **"Coding for Bio: Linux and BASh"**.


The first project involves basic tasks such as:


- Creating a directory and displaying its contents
- Downloading files and moving them across directories
- Searching through a text file using the commands **"awk"** and **"grep"**
- Viewing a history of commands used and clearing the terminal space.
